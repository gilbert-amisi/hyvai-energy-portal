# ⚡ HYVAI Energy Web Portal

A modern, full-stack web application developed for **HYVAI Energy**. This project integrates a powerful headless CMS with a reactive frontend to deliver high performance and scalable energy data management.

---

## 🏗️ Architecture & Tech Stack

This project follows a **decoupled architecture** (Headless), separating the data management from the user interface.

* **Frontend:** [Next.js](https://nextjs.org/) (React) – Leveraging Server-Side Rendering (SSR) for optimal SEO and speed.
* **Backend:** [Strapi](https://strapi.io/) (Headless CMS) – Providing a flexible API and an intuitive admin panel.
* **Database:** PostgreSQL – Ensuring data integrity and production-ready performance.
* **Styling:** Tailwind CSS – For a clean, responsive, and modern user interface.

---

## 📂 Project Structure

- `/frontend`: The Next.js application, including components, pages, and API integration logic.
- `/backend`: The Strapi CMS configuration, content types, and business logic.
